# ❄️ Climatech Full Stack

Sistema web desenvolvido para empresas do segmento de climatização, oferecendo presença digital profissional e canais de contato para captação de clientes.

---

## 🌐 Sobre o Projeto

O Climatech Full Stack foi desenvolvido para apresentar serviços de instalação, manutenção e assistência técnica em sistemas de climatização.

O projeto possui frontend responsivo e backend para gerenciamento das informações da plataforma.

---

## ✨ Funcionalidades

- Página inicial moderna
- Apresentação de serviços
- Sobre a empresa
- Formulário de contato
- Layout responsivo
- Integração com WhatsApp
- Navegação otimizada

---
